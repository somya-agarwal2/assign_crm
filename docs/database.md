# Database Architecture

EngageX uses SQLAlchemy as its ORM. In development, it uses SQLite, and in production, it is configured for PostgreSQL.

The core architecture revolves around a strict **Multi-Tenant Model**. Every major entity contains a `workspace_id` foreign key. This ensures data isolation at the query level across different enterprise clients.

## Core Entities

### Organization & Workspace
- **Organization**: The top-level billing or enterprise entity.
- **Workspace**: A partitioned workspace under an organization. Users log into a workspace. All CRM data (Customers, Segments, Campaigns) are scoped to a workspace.
- **User**: A staff member with a specific role (`Admin`, `Marketing Manager`, `Analyst`) mapped to a workspace.

### CRM Layer
- **Customer**: A unified profile tracking behavioral metadata (total spent, order count, churn score) and PII.
- **Order / OrderItem**: E-commerce or transactional data mapped back to the customer to compute LTV and recent purchases.

### Marketing Layer
- **Segment**: A dynamic or static audience subset. Uses JSON `filters` to define query rules. Maps to Customers via `segment_customers` association.
- **Campaign**: A bulk or personalized message broadcast. Contains the logic for the `offer` and expected revenue.
- **Template**: Reusable HTML/Text message blocks used by Campaigns or Journeys.

### Journey Engine
- **Journey**: A multi-step automation workflow.
- **JourneyNode**: The individual steps inside a journey (Trigger, Action, Condition, Wait).
- **JourneyRun**: Tracks an individual customer passing through a Journey.
- **JourneyLog**: The execution history of a customer within a JourneyRun (e.g., "Email Sent at 12:00").

### AI & Analytics
- **AIOpportunity / AICampaignOpportunity**: Predictive recommendations generated by the backend AI jobs.
- **AnalyticsSnapshot**: Time-series rollups for the main dashboard (Revenue, Conversions, Growth).
- **DeliveryEvent**: Granular delivery state (Sent, Opened, Clicked, Converted) tracked per campaign and customer.
